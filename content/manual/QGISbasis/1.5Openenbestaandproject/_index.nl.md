Indien er al een project is aangemaakt (door jezelf of door een collega) kan dit project ook geopend worden door alle andere gebruikers. Bij het openen vanuit de “verkenner” gaat dit hetzelfde als voor ieder ander soort bestand (Word, Excel, etc.) door er dubbel op te klikken. Qgis gaat daarna opstarten en het project wordt direct geladen. Hierbij worden alle datalagen die in het project waren opgeslagen ook geladen.



De andere mogelijkheid voor het openen van een QGIS-project is door deze vanuit QGIS zelf te gaan openen. Dit kan op drie verschillende manieren.



<p><strong>Methode 1: openen van recente projecten vanuit het opstartscherm</strong></p>

<div style="margin-left: 1em;">

1. In blok ‘2’ van Figuur 1.3 is een lijst beschikbaar met alle recent geopende projecten van jouw computer.

2. Door te dubbelklikken op een recent project gaat QGIS dit project openen, de werkwijze is vergelijkbaar als voor het openen van een sjabloon. 
</div>


<p><strong>Methode 2: openen recent project vanuit taakbalk</strong></p>

<div style="margin-left: 1em;">

1. In Figuur 1.4 wordt links bovenin op “project’ geklikt (blok ‘1’ in Figuur 1.3), er opent nu een dropdown menu.

2. In het dropdown menu is de derde optie “recent geopend”, er komt een extra lijst aan opties zichtbaar.

3. In de volledige lijst zijn alle recent geopende projecten zoals die via methode 1 ook beschikbaar waren zichtbaar.

4. Bij het klikken op een project opent het desbetreffende project zich.
</div>


<p><strong>Methode 3: openen bestaand project vanuit taakbalk</strong></p>

<div style="margin-left: 1em;">

1. In Figuur 1.4 wordt links bovenin op “project’ geklikt (blok ‘1’ in Figuur 1.3), er opent nu een dropdown menu.

2. In het dropdown menu is de tweede optie “openen”, er opent zich een pop-upscherm met de verkenner.

3. Je kunt nu naar de locatie waar het project is opgeslagen en het bestand selecteren.

4. Nadat het bestand is geselecteerd klik je in het pop-scherm op ‘openen’, QGIS gaat het project nu openen.
</div>
